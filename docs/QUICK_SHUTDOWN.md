# 🛑 ARRÊT RAPIDE - Guide Rapide

## Le plus important

**Quand vous voulez arrêter le serveur :**

```
👉 Appuyez une fois sur CTRL+C dans la console
```

**C'est tout !** ✅ (Un seul CTRL+C suffit)

---

## Que se passe-t-il ?

```
╔════════════════════════════════════════════════════════════╗
║  SHUTDOWN SIGNAL RECEIVED - Saving data before closing...  ║
╚════════════════════════════════════════════════════════════╝

Step 1/4 - Stopping new connections...          (rapide)
Step 2/4 - Saving world data...                  (2-5 sec)
Step 3/4 - Disconnecting players...              (1-3 sec)
Step 4/4 - Closing database connections...       (0.5 sec)

✅ SERVER SHUTDOWN COMPLETE - All data saved

[La console se ferme automatiquement]
```

---

## ✅ Données SAUVEGARDÉES

- 👤 Tous les joueurs
- 💰 Tous les items/équipements
- 🏠 Toutes les maisons
- 🐴 Tous les montures
- 🎁 Tous les objets en jeu
- 📦 Tous les comptes
- ⚔️ Toutes les guilde

---

## ⏰ Durée

**3 à 10 secondes** maximum (puis fermeture automatique)

---

## ❌ NE PAS FAIRE

```
❌ Alt+F4 (fermer brutalement) - Au lieu de CTRL+C
❌ Tuer le process Java - Au lieu d'utiliser CTRL+C
❌ Débrancher l'électricité
❌ Forcer l'arrêt Windows
```

---

## ✅ À FAIRE

```
✅ CTRL+C (une seule fois)
✅ Attendre les 4 étapes (2-10 secondes)
✅ La console se ferme automatiquement
✅ C'est fini !
```

---
