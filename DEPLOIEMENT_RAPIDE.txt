⚠️ EN CAS DE PROBLÈME
──────────────────────────────────────────────────────────────────

PROBLÈME: Le serveur ne démarre pas
SOLUTION:
  1. Vérifier les logs: Logs/Error/*.log
  2. Vérifier que Java est bien installé
  3. Vérifier que le port est libre
  4. Restaurer l'ancienne version si nécessaire

PROBLÈME: Les IA ne jouent toujours pas
SOLUTION:
  1. Vérifier les logs de démarrage
  2. Vérifier qu'il y a des monstres sur la map
  3. Vérifier que la version MySQL est bien 9.4.0
  4. Redémarrer le serveur complètement

PROBLÈME: Erreur MySQL connection
SOLUTION:
  1. Vérifier que le service MySQL est actif
  2. Vérifier les identifiants MySQL
  3. Vérifier la base de données existe
  4. Vérifier le driver MySQL 9.4.0 est chargé

RESTAURATION: Revenir à la version précédente
SOLUTION:
  1. Copier l'ancien JAR
  2. Redémarrer le serveur
  3. Tester les logs
